[ 샘플 DB 설치하기 ]
1. North  Wind라는 샘플 DB  설치하기
2. Northwind DB 소개
    - MS Access 프로그램의 기본 DB(2010년 버전)
    - 설치 기본형은 mySQL용
    - Northwind는 고객, 주문, 재고, 공급업체, 배송 등의 소규모 비지니스용 DB
3. google검색으로 Northwind 쿼리문 찾기
    -> "mysql Northwind sample database" 검색
    -> 결과 : 첫번째 결과 링크 -> dalers/mywind... 깃허브 사이트
    -> Northwind.sql : 테이블 생성 쿼리문
    -> northwind-data.sql : 데이터 삽입 쿼리문
4. 쿼리문을 메인 SQL에서 실행하면 "northwind" DB가 만들어지고 거기에 각 테이블이 생성된다
    -> [ northind DB 깃허브 ]
